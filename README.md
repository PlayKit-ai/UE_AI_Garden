# PlayKit Unreal SDK Sample Project - AI Garden

**Made with ❤️ by the Playkit.ai Team**

An UE5.6.1 Game that uses PlayKit Unreal SDK. This Sample project utilizes chat, image generation and runtime 3d generation. 

An idle decoration game where imagination becomes reality. Collaborate with AI via chat to generate exclusive 3D collections under completely free rules, weaving a legendary tale for every one-of-a-kind.
